# JavaScript Exercises

This folder contains exercises and small projects developed during the JavaScript module of the Frontend Specialization course.

✨ Core Concepts
- JavaScript Fundamentals

🔐 Web and Security

- Cookies and Web Storage

- DOM Manipulation (create, copy, move, delete elements)
- AJAX (concept of asynchronous data transmission)

⚙️ Advanced Topics
- Frameworks and Creating Your Own Library
- Search Engine Optimization (SEO)
- Single Page Applications (SPA)
